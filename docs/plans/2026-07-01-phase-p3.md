# Codebase Atlas Phase 3 (Reasons Pane) тАФ Task Plan

Generated: 2026-07-01
AC coverage: US-011..US-014 (plus cross-cutting: PaneTabs, validation, failure, cache-corruption)
NFRs in scope: COST-001/002, FRESH-001, QUAL-001, SEC-002, MAINT-002, USABILITY-001, AVAIL-001
Estimated waves: 9
Parallelizable tasks: limited (solo, sequential) тАФ dependency-ordered
Depends on: P1 shipped (TypedGraphModel, SelectionState, DetailPane shell)
Contract: P3 owns `PaneTabs`/`TabId`/`TabRegistration`, `ViewState.activeTab`, `checkStaleness` generic, `TradeoffMatrix.id` slug contract.

## Anti-patterns to watch
- LLM egress leaking beyond the injected LLMClient/SealTestClient (SEC-002) тАФ Task 10 guards.
- Uncached re-synthesis on every render (COST-001 violation) тАФ Task 4 + Task 11 guard.
- Hand-authoring matrices instead of synthesizing (ADR-004 fallback, not the chosen path) тАФ do not drift.
- ReasonsPane prose bloating cells (COST-002 тАФ compress words, never reasoning) тАФ Task 7 guards.
- Right-pane tab chrome built P3-only and incompatible with P4 тАФ P3 owns `PaneTabs`; P4's `RepresentationContent` registers tabs via the `TabRegistration` API.
- LLM call without opt-in (SEC-002) тАФ Task 0a config + opt-in gate.
- LLM failure / timeout unhandled (causes crash) тАФ Task 3b (adapter), Task 3 (synth), Task 11 (controller) tests.
- Corrupt cache file crashes the app тАФ Task 4 (`load` tolerant).
- Incomplete entries rendered as complete matrices (QUAL-001 violation) тАФ Task 7 (ReasonsPane), Task 4 (cache.put refuses), Task 11 (controller skips) tests.
- Defining `TabId` / `RepresentationHint` twice тАФ P3 owns `TabId`; P4/P5 import from `PaneTabs`.
- Duplicating the freshness function тАФ P3 exports `checkStaleness`; P5 imports it; do not redefine.

## Invariants enforced by tests
1. **Synthesizer calls LLM at most once per decision per doc-hash** (COST-001) тЖТ Task 4.
2. **Seal-test confidence >= 0.8 to pass** (QUAL-001) тЖТ Task 6.
3. **Render serves cache hit with zero LLM calls** (COST-001) тЖТ Task 11.
4. **Core (synth/cache/seal-test) imports nothing from React** тЖТ Task 10.
5. **TradeoffMatrix binds only to existing AtlasNode/community ids** (no dangling refs) тЖТ Task 2.
6. **Matrix cells <= word cap** (COST-002) тЖТ Task 2.
7. **Same decision doc тЖТ same `TradeoffMatrix.id` (deterministic slug)** тЖТ Task 2.
8. **LLM egress is ONLY through the configured adapter; opt-in required; no hardcoded URLs** тЖТ Task 0a, 3b, 10.
9. **Incomplete entries never render as a complete matrix** (QUAL-001 enforcement) тЖТ Task 4 (cache.put refuses), Task 7 (ReasonsPane renders incomplete state), Task 11 (controller skips).
10. **`PaneTabs` exports `TabId` and `TabRegistration`; P4/P5 import from here** тЖТ Task 7a.

## Conventions in scope
Run `/conventions init` (or extend existing) after Task 7 lands to capture the LLM-injection + cache-file + PaneTabs patterns.
